# 📱 Yuzi Phone 玉子手机

SillyTavern 独立手机扩展，提供手机主屏、消息记录表实时回复、AI 指令预设、专属/通用表格视图与设置管理能力。

## 功能

- 📱 手机容器：支持悬浮显示、拖拽移动、缩放与位置记忆
- 🏠 主屏与 App：提供主屏入口、表格查看与设置页面
- 💬 实时回复：可在消息记录表内直接调用 AI 生成角色回复
- 🧠 AI 指令预设：支持分段提示词、主槽位排序、预设导入导出与占位符注入
- 📚 上下文控制：会话标题、世界书、正文上下文、历史消息共同参与回复生成
- 🖼️ 结构化媒体：支持“正文 / 图片描述 / 视频描述”三段式回复并写回表格
- 🎨 专属视图美化：覆盖消息、动态、论坛三类专属界面与通用模板渲染
- ⚙️ 数据与设置：支持世界书桥接、聊天配置、独立命名空间存储

## 安装说明

### URL 安装

1. 打开 SillyTavern
2. 进入“扩展” → “安装扩展”
3. 输入仓库地址：`https://github.com/yuzi83/st-yuzi-phone`
4. 安装后重启 SillyTavern

### 手动安装

1. 下载项目压缩包或最新 Release
2. 解压到 `SillyTavern/public/scripts/extensions/third-party/st-yuzi-phone`
3. 重启 SillyTavern
